In English:

This remote repository is for my own personal HTML5 game experiments and exercises.
Some could be purely made with nothing HTML5, CSS and Javascript while others could 
be made with existing frameworks and engines.

Either way, I'm creating these to learn more about game development and also to slowly
fill holes in my knowledge.

Suomeksi:

Tämä tietovarasto sisältää omia HTML5 pelikokeiluja ja harjoitteita. Jotkin on saatettu
tehdä puhtaasti vain HTML5, CSS ja Javascriptillä ja toiset taas frameworkien ja 
pelimoottoreiden avulla.

Joka tapauksessa, teen näitä oppiakseni lisää pelikehityksestä ja täydentääkseni tietojani.
